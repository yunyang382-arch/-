# 关于当前股市运动规律与我们的战术方针

（二〇二六年一月）

当前，证券市场内的斗争形势错综复杂，行情起伏剧烈，犹如“过山车”一般。许多同志对于这种“反复震荡”的局面感到不适应，产生了一些糊涂思想。有的盲目乐观，认为大军可以直接长驱直入，攻占四千点高地；有的则悲观失望，一遇挫折就想“清仓跑路”。这两种倾向都是不对的。

为了澄清思想，统一认识，我们必须运用辩证唯物主义的观点，深入分析主力资金（即所谓“龙哥”）的作战规律，认清市场所处的发展阶段，并据此制定我们正确的战略战术。

### 一、 论主力资金的作战规律

我们研究主力资金，必须从“大操作”和“微操作”两个层面来看，切不可被表面的假象所迷惑 。

**第一，战略上的“大操作”，主要表现为底部的果断介入。**
历史经验告诉我们，主力往往在市场极度恐慌、众人皆言“股市要倒闭”的危急时刻，集中优势兵力，通过宽基指数（如沪深300、上证50）进行战略性建仓。二〇二四年的几次关键护盘，以及九二四行情的启动，分时图上的放量脉冲，就是他们进场的铁证。这种操作是大手笔，意在稳住阵脚，确立多头根据地 。

**第二，战术上的“微操作”，主要表现为对“工具”的灵活运用。**
银行、保险、证券，这“三驾马车”，在当前不仅仅是投资标的，更是调节市场温度的**工具**。

* 当市场情绪过热、斜率过高（如八月份科技股疯涨）时，主力便拉升银行股，名为护盘，实为“吸血”降温，防止市场变成脱缰的野马 。


* 当市场大跌、人心浮动（如十月四日前后）时，主力又拉升权重股，进行托底，维持行情的体面 。


* 至于证券板块，主力采取的是“进二退一”的游击战法。大单敲入后，往往会让股价自然回落，甚至连续调整数日，以降低冲击成本。同志们如果看到券商大涨就盲目追高，往往会陷入敌人的包围圈；正确的做法应当是敌疲我打，在回调时寻找战机 。



### 二、 论市场发展的三个阶段

纵观A股历史，市场运动大致存在三种状态 ：

* 
**第一种，是“自由野马”的状态。** 如二〇一五年的疯牛和二〇二一年的创业板，无人管束，情绪主导，一哄而上，最后必然导致泡沫破裂，一地鸡毛。这是散户最喜欢的，但也是最危险的 。


* 
**第二种，是“强力管控”的状态。** 如二〇一六年至二〇一七年的“漂亮50”行情，主力控盘严密，走出一种压抑的“慢牛”。这是“灾后重建”时期的特殊产物 。


* **第三种，是“牵绳遛狗”的状态，也就是我们现在的状态。**
这根绳子掌握在主力手中。当市场这只“狗”跑得太快、太疯时（斜率过高），主力就会收紧绳子，打压降温；当“狗”跑不动、情绪低迷时，主力就会放松绳子，甚至扔根骨头（拉升权重），诱导其前行。
我们必须清醒地认识到，监管层的意图是**“慢牛”**，绝不允许二〇一五年式的“疯牛”重演。任何试图挑战这一底线的行为，都会遭到政策的“当头棒喝” 。



### 三、 实事求是地衡量市场温度

目前市场的热度究竟如何？如果我们以二〇一五年疯牛顶峰为一百分，现在的热度大约在**八十分**左右 。

我们不仅要看指数，还要看结构：

1. 
**从杠杆资金看：** 融资余额占流通市值的比例尚在安全区间（约2.5%），远未达到二〇一五年的警戒线（6%）。但融资买入额占成交量的比例已经亮起红灯，说明局部过热 。


2. 
**从参与主体看：** 并没有出现大学生集体开户、全民炒股的盛况。目前的资金流动主要是“大户搬家”，而非散户的全面进场。基民对基金经理的信任尚未恢复，这与二〇二一年有着本质区别 。


3. 
**从赚钱效应看：** 现在的涨跌比约为二比三，即便指数在涨，亏钱的人依然不少。这说明“普涨”的时代已经过去，结构性矛盾依然突出 。



### 四、 我们的战略部署

基于上述分析，我们应当采取什么样的策略？

**第一，控制仓位，留有余地。**
建议以**七成仓位**作为分界线。激进的同志可以略高，稳健的同志可以略低，但务必保留三成左右的流动力量。这既是为了在急跌时有能力补仓，也是为了保持心态的平稳，避免犯“左”倾冒险主义的错误 。

**第二，灵活机动，反对教条主义。**
不要死守一个板块。科技股固然是主攻方向，但也要配置一定的防御性力量（如红利、中字头、消费）。这叫作“统一战线”，东方不亮西方亮。当某一板块涨幅过大时，要学会“滚动操作”，不要贪婪地想吃尽最后一段“鱼尾” 。

**第三，丢掉幻想，准备持久战。**
不要指望主力会发善心，也不要指望市场会单边上涨。过山车行情才刚刚开始，震荡将是常态。我们要在震荡中寻找机会，积小胜为大胜。

同志们，道路是曲折的，但前途是光明的。只要我们掌握了主力的规律，控制好自己的贪欲和恐惧，就一定能在这场复杂的市场博弈中，取得最后的胜利！
