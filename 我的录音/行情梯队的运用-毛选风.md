# 关于十月战役的战略部署与梯队作战问题

（二〇二六年一月）

当前，市场正处于一个重要的转折关口。九月的行情，磨磨蹭蹭，不上不下，导致许多同志产生了悲观情绪，信心不足。这是不对的。我们必须清醒地认识到，十月将是变盘之月，无论是向上还是向下，有波动就比死水一潭要好。

为了迎接即将到来的战斗，我们有必要对当前的敌我形势、主力资金（所谓“龙哥”）的意图以及科技股作战的梯队战法，进行一次彻底的分析。

### 一、 论“管制”与“自由”的辩证关系

目前市场的主要矛盾，是**监管层的“慢牛”意图与市场资金追求“疯牛”本能之间的矛盾** 。

所谓“龙哥”，代表的是一种管制的逻辑。他们希望股市稳步上涨，服务于国家经济和居民财富增值，不希望重演二〇一五年的悲剧，因此要压制节奏，搞“稳台阶” 。

然而，市场自有其规律。当做多的力量积蓄到一定程度，管是管不住的，这就叫“自由的步伐”。作为一名革命的股民，我们当然渴望这种自由的狂奔，但现阶段，我们必须服从于“管制”的现实。我们要在管制的缝隙中寻找机会，既不要盲目冲动，也不要妄自菲薄。

### 二、 集中优势兵力，运用“三级梯队”战法

在科技股这个主战场上，我们必须讲究战术，绝不能搞“一锅煮”。要把科技股划分为三个梯队，采取灵活机动的战略战术 。

**第一梯队，是我们的“主力军”。**
这部分力量启动最早（六七月份），基本面最硬，如中际旭创、胜宏科技以及海外的美光、腾讯等。它们是行情的发起者，但目前已显疲态，正在进行战略休整。对于这部分力量，在它们休息的时候，我们不能死守，要敢于把兵力调拨出去 。

**第二梯队，是当前的“突击队”。**
这部分力量爆发于八月，如寒武纪、海光信息等。它们爆发力强，弹性大，是目前战斗最激烈的地方。当主力军休整时，我们就要集中兵力，投入到这部分突击队中去，打歼灭战 。

**第三梯队，是后备的“游击队”。**
这部分力量启动最晚（九月以后），多是些讲故事、蹭热点的小票，如江波龙等。它们的特点是“来得快去得快”，纯属情绪博弈。对于这部分，我们可以搞，但要速战速决，打得赢就打，打不赢就跑 。

**我们的作战方针是：** 在第二梯队出现前，死守第一梯队；当第二梯队爆发时，果断切换战场；当所有故事都讲完、游击队溃散时，资金必将回流，去解救第一梯队的主力。这就是“围魏救赵”，这就是资金的轮回 。

### 三、 认清A股科技的本质，实行“高低切换”的统一战线

我们必须实事求是地指出：**A股科技股的估值，是“不可描述”的。** 相比于美股、港股科技巨头二三十倍的市盈率，我们很多科技股动辄几百倍甚至亏损，纯粹是靠讲故事。这是典型的“纸老虎”。因此，我们做A股科技，只能做趋势，做投机，千万不能当真做价值投资 。

鉴于科技股高位拥挤且面临监管压力，我们必须开辟第二战场，实行**“高低切换”**。要把目光投向那些低位的、反内卷的传统行业（如化工、金属、建材）以及政策预期的方向（如大金融、消费、地产） 。这叫作“农村包围城市”，利用板块轮动，保存实力，消灭敌人。

### 四、 丢掉幻想，守住阶层

最后，谈一谈阶层跃迁的问题。
过去四十年，我们经历了下海经商、房地产爆发两次大的历史机遇。现在的资本市场，可能是普通人面临的第三次，也是极为罕见的一次机遇。

但是，道路是曲折的。对于大多数普通同志来说，不要妄想一夜暴富跨越阶层，那是极少数人的神话。能够在这个残酷的市场中，守住父辈留下的基业，稳住自己当前的台阶，就已经是一种伟大的胜利了 。

同志们，十月的号角已经吹响。我们要保持清醒的头脑，运用梯队战法，灵活应变。既要看到光明的前途，也要准备经受风浪的考验。胜利终将属于理性的投资者！
